# Backend (Hono)

Dieser Ordner enthält die API/Backend-Anwendung, betrieben mit Hono und Node.js.
Hier ist die Haupt-API-Schnittstelle definiert, die per TypeScript geteilt oder direkt aufgerufen wird.

## Struktur

- `src/`: Enthält Hono-Routen, Controller, und Validierung via Zod.

## Befehle

- `pnpm dev`: Startet den Hono-Dev-Server (via tsx).
- `pnpm build`: Kompiliert das Backend.
- `pnpm lint`: Führt ESLint aus.
